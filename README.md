## Módulo em Elixir representando uma máquina de venda.

* A cada R$ 0,45, um refrigerante é recebido
* Somente valores de R$0,45, R$ 0,10 ou R$ 0,25 são aceitos
* Caso insira mais, o valor será "comido"
